# Vue2 com Nuxt

### Passo a passo
Clone Repositório
```sh
git clone https://github.com/gestaobytes/evoterra-frontend evoterra
```

```sh
cd evoterra/
```

## Build Setup

``` bash
# instalar as dependencias
$ yarn install

# servido com hot reload: localhost:3000
$ yarn dev

# gerar arquivos para produção rodar em servidor
$ yarn build
$ yarn start

# gerar páginas estáticas
$ npm generate
```
